# Cat's Cradle — Neon Hand Strings

An interactive, AI-powered neon art experience that transforms your hands into a digital cat's cradle. Using MediaPipe for high-performance hand tracking and p5.js for fluid generative rendering, this project creates a glowing, reactive playground in your browser.

## ✦ Features

- **Dynamic String Generation**: Move your hands to weave neon strings that react to tension and proximity.
- **Interactive Gestures**:
  - **Constellation Mode**: Flash a peace sign to connect all finger joints into a geometric constellation.
  - **Pinch Burst**: Pinch your thumb and index finger to trigger golden particle explosions.
- **Ambient Environment**: A reactive starfield and particle system that swirls around your movements.
- **Retro Aesthetic**: Toggle a CRT scanline overlay for a classic cybernetic feel.
- **Performance Optimized**: Automatically scales visual quality based on your system's frame rate.

## ✦ Controls

- **S**: Toggle Scanline Effect
- **F**: Toggle FPS Counter
- **Drag Camera**: Move the live camera preview anywhere on the screen.
- **Collapse Camera**: Minimize the camera preview to stay focused on the art.

## ✦ Technical Stack

- **MediaPipe Hands**: Real-time hand landmark detection.
- **p5.js**: Creative coding library for the visual simulation.
- **Vanilla CSS/JS**: High-performance UI and state management.

## ✦ Getting Started

Simply open `index.html` in any modern web browser with webcam access. 

*Note: For the best experience, ensure you are in a well-lit environment.*
